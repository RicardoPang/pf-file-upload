## Big File

### 前端启动

- 拉项目

  ```bash
  git clone git@github.com:Devil-Training-Camp/pf-file-upload.git
  ```

- 执行 npm install

  ```base
  npm install
  ```

- 运行项目

  ```
  npm run dev
  ```

### 服务端目启动

- 进入目录

  ```bash
  cd server
  ```

- 执行 npm install

  ```base
  npm install
  ```

- 运行项目

  ```
  npm run dev
  ```
